# 엔드이펙터 속성 db

**열:** `필드키` · `한글명` · `타입` · `단위` · `형식` · `속성노출` · `기호` · `tab` · `설명` · `우선순위` · `비고`  
(원본의 **노출 여부**와 **속성 노출** 값이 행마다 동일하여 `속성노출` 한 열로 통합.)
